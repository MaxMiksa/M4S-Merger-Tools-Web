# Release Notes / 版本说明

## v1.0.0 – Web Platform Launch

## ✨ 网页版首发：随时随地，即点即用

**M4S 合并工具现已推出网页版！无需下载任何软件，无需安装 FFmpeg，打开浏览器即可在本地合并您的 Bilibili 缓存文件。**

| 类别 | 详细内容 |
| :--- | :--- |
| **零安装** | 利用 WebAssembly 技术，直接在浏览器运行核心引擎，数据不上传服务器，安全且隐私。 |
| **现代化设计** | 全新的“Modern SaaS”设计语言，支持深色/浅色/跟随系统主题，磨砂玻璃质感，视觉体验极佳。 |
| **灵活合并** | 支持仅合并视频、仅合并音频或音视频混流。智能识别输入，自动调整处理逻辑。 |
| **双语支持** | 界面与日志全流程支持中文与英文切换，满足不同用户需求。 |

## ✨ Web Launch: Merge Anywhere, Instantly

**M4S Merger Tool is now available on the web! No software downloads, no FFmpeg installation required. Merge your M4S segments locally directly in your browser.**

| Category | Details |
| :--- | :--- |
| **Zero Installation** | Powered by WebAssembly, the core engine runs directly in your browser. No data is uploaded to servers—secure and private. |
| **Modern Design** | Brand new "Modern SaaS" design language featuring Light/Dark/System themes and glassmorphism for a premium visual experience. |
| **Flexible Merging** | Supports merging video-only, audio-only, or muxing both. Smartly detects inputs and adjusts processing logic automatically. |
| **Bilingual Support** | Full interface and log support for both Chinese and English. |
