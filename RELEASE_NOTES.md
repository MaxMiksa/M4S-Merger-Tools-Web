# Release Notes / 版本说明

## v2.0.0 – Performance & Reliability Update / 性能与可靠性升级 (2025年12月6日)

## ✨ 智能极速混流，兼顾稳定与高效

**M4S 合并工具在保持易用性的前提下，对核心混流引擎进行了重大升级，大幅提升了处理速度，并引入智能兼容性保障。**

| 类别       | 详细内容                                                     |
| :--------- | :----------------------------------------------------------- |
| **极速性能** | 默认优先尝试“全流复制”模式，对于常见的 M4S 文件，混流速度可达秒级，极大缩短等待时间。 |
| **智能兼容** | 引入智能检测与回退机制。若极速模式因文件格式不兼容而失败，系统将自动切换到音频重编码模式，确保所有文件都能顺利合并，兼顾速度与稳定性。 |
| **优化体验** | 用户无需关心底层技术细节，即可享受到更快、更可靠的混流体验。 |

## ✨ Intelligent & Ultra-Fast Merging with Enhanced Reliability

**The M4S Merger Tool receives a significant upgrade to its core merging engine, delivering substantial speed improvements and introducing intelligent compatibility safeguards, all while maintaining its user-friendly design.**

| Category          | Details                                                        |
| :---------------- | :----------------------------------------------------------- |
| **Ultra-Fast Performance** | Now prioritizes "full stream copy" mode. For most common M4S files, merging can be completed in seconds, drastically reducing waiting times. |
| **Intelligent Compatibility** | Implements smart detection and fallback. If the ultra-fast mode fails due to file format incompatibility, the system automatically switches to an audio re-encoding mode, ensuring all files can be successfully merged, balancing speed with stability. |
| **Optimized Experience** | Users can now enjoy a faster and more reliable merging experience without needing to worry about the underlying technical complexities. |

## v1.0.0 – Web Platform Launch

## ✨ 网页版首发：随时随地，即点即用

**M4S 合并工具现已推出网页版！无需下载任何软件，无需安装 FFmpeg，打开浏览器即可在本地合并您的 Bilibili 缓存文件。**

| 类别 | 详细内容 |
| :--- | :--- |
| **零安装** | 利用 WebAssembly 技术，直接在浏览器运行核心引擎，数据不上传服务器，安全且隐私。 |
| **现代化设计** | 全新的“Modern SaaS”设计语言，支持深色/浅色/跟随系统主题，磨砂玻璃质感，视觉体验极佳。 |
| **灵活合并** | 支持仅合并视频、仅合并音频或音视频混流。智能识别输入，自动调整处理逻辑。 |
| **双语支持** | 界面与日志全流程支持中文与英文切换，满足不同用户需求。 |

## ✨ Web Launch: Merge Anywhere, Instantly

**M4S Merger Tool is now available on the web! No software downloads, no FFmpeg installation required. Merge your M4S segments locally directly in your browser.**

| Category | Details |
| :--- | :--- |
| **Zero Installation** | Powered by WebAssembly, the core engine runs directly in your browser. No data is uploaded to servers—secure and private. |
| **Modern Design** | Brand new "Modern SaaS" design language featuring Light/Dark/System themes and glassmorphism for a premium visual experience. |
| **Flexible Merging** | Supports merging video-only, audio-only, or muxing both. Smartly detects inputs and adjusts processing logic automatically. |
| **Bilingual Support** | Full interface and log support for both Chinese and English. |
