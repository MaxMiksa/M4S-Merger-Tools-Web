# （示例样品） Release Note / 版本说明

本文档记录每次发布时给用户的双语摘要，强调通俗、重点明确。每次更新都在顶部追加最新版本条目。
> 规则：尽量使用日常语言、避免过于专业的缩写；中英文表格行列一致，覆盖本次版本最重要的 3–4 个点即可。
功能和UI的优化可以分类分开说。

## 模板 / Template

```
# Version X.Y.Z 

## ✨ 中文亮点标题

一句简单中文描述，说明本次更新解决了什么问题或提升。

| 类别 | 详细内容 |
| --- | --- |
| 项目 1 | 简短中文说明 |
| 项目 2 | 简短中文说明 |

## ✨ English Highlight Title

A short English sentence explaining what this release improves or fixes.

| Category | Details |
| --- | --- |
| Item 1 | Short English description |
| Item 2 | Short English description |
```

---

## 示例

```
# Version 1.1.0 

## ✨ UI/UX 大幅改进与多语言支持

**本次更新对用户界面和体验进行了全面彻底的重构，使得工具更加友好、现代化和可靠。**

| 类别 | 详细内容 |
| :--- | :--- |
| **界面设计与体验** | UI 完全更新：采用更现代、更直观的设计，操作更方便、更友好、更可靠。 |
| **多语言支持 (I18N)** | 中英文无缝切换：软件界面支持中英文双语实时切换。 |
| **双语提示与日志** | 双语提示支持：安装过程中的异常问题提示支持中英双语显示；文件处理过程中的日志支持实时双语显示。 |
| **显示模式** | 主题切换：软件界面支持浅色 (Light) 和深色 (Dark) 模式切换，适应不同用户的视觉偏好。 |

## ✨ Major UI/UX Overhaul and Multi-language Support

**This update introduces a complete and thorough restructuring of the user interface and experience, making the tool more friendly, modern, and reliable.**

| Category | Details |
| :--- | :--- |
| **Interface Design & UX** | Complete UI Refresh: Updated to a more modern, intuitive design, resulting in easier, friendlier, and more reliable operation. |
| **Multi-language Support (I18N)** | Seamless Language Switching: The software interface now supports real-time, seamless switching between Chinese and English. |
| **Bilingual Prompts & Logs** | Bilingual Feedback: Exception prompts during the installation process, as well as real-time logs during file processing, are now displayed bilingually (Chinese and English). |
| **Display Mode** | Theme Switching: The software interface supports switching between Light and Dark display modes to accommodate user preference. |
```
